# Develop Environment
1. JDK 1.8 +
2. Maven 3.5 + 
3. IntelliJ IDEA ULTIMATE 2021.3.3+

# How to run application

1. Clone the project

   `git@github.com:N3ov/innova-project.git`

2. IDEA open the project file

3. Find `pom.xml` and download dependency from `Maven`

4. Run the `ProjectApplication`
